# Python Flask Demo Project

这是一个简单的 Python Flask Web 应用，用于测试部署平台对 Python 项目的支持。

## 项目结构
- `app.py`: Web 服务器主文件 (监听 8000 端口)
- `requirements.txt`: 依赖列表 (包含 Flask)

## 如何部署
1. 将当前文件夹 (`demo-project-python`) 的内容压缩为 ZIP 文件（确保 `app.py` 位于 ZIP 根目录）。
2. 在部署平台首页选择 **"上传 ZIP 压缩包"**。
3. 上传该 ZIP 文件。
4. 平台应会自动识别为 **Python** 项目。
5. 点击部署，等待完成。
